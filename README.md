
Интерактивный калькулятор, который помогает рассчитать итоговую стоимость заказа с учётом выбранного товара, количества, дополнительных услуг и промокода.

## 🛠 Стек технологий

- HTML5
- CSS3
- JavaScript (ES6+)

## ✨ Функционал

- Выбор товара из выпадающего списка (цена подставляется автоматически)
- Указание количества (минимум 1)
- Чекбоксы для дополнительных услуг (гарантия, доставка, упаковка)
- Поле для ввода промокода (доступен код `SKIDKA10` — скидка 10%)
- Динамический расчёт итоговой суммы
- Валидация количества и вывод ошибки при некорректном вводе
